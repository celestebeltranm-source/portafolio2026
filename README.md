# Portfolio Celeste Beltrán

## Cómo usar

### Subir a Netlify (recomendado)
1. Ve a https://app.netlify.com/drop
2. Arrastra esta carpeta completa
3. Tu sitio queda publicado en segundos

### Subir a GitHub Pages
1. Sube todos los archivos a un repo en GitHub
2. Settings → Pages → Branch: main → / (root)

---

## Flujo de edición

1. Abre `index.html` en tu navegador (doble clic)
2. Ingresa con tu contraseña
3. Edita textos, colores e imágenes
4. Haz clic en **"Guardar y publicar"**
5. Se descarga un nuevo `index.html` — reemplaza este archivo
6. Sube la carpeta de nuevo a Netlify

## Estructura
```
portfolio-celeste-sitio/
├── index.html      ← Tu portafolio completo (todo embebido)
├── _redirects      ← Necesario para que las URLs funcionen en Netlify
└── README.md       ← Este archivo
```

## URLs del sitio
- `/` → Home
- `/#casos` → Casos de Estudio
- `/#proyectos` → Proyectos
- `/#corner` → Design Corner
- `/#cv` → CV
- `/#sobre` → Sobre mí
- `/#contacto` → Contacto
